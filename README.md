# Projet Java – Application de Jeux de Cartes

##  Présentation générale
Ce projet consiste à développer une application de jeux de cartes en Java.
Il a été réalisé dans un cadre académique afin de mettre en pratique les concepts
fondamentaux de la programmation orientée objet (POO).
